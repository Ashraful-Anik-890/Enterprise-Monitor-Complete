📸 Screenshots POST multipart/form-data
📋 Demo Format (multipart — NOT JSON) 
Content-Type: multipart/form-data

Form fields:
  pcName        = "DESKTOP-ABC123"
  macAddress    = "00:1B:44:11:3A:B7"
  userName      = "jdoe"
  timestamp     = "2026-02-22T08:30:00+00:00"
  activeWindow  = "Google - Chrome"
  activeApp     = "chrome.exe"
  syncTime      = "2026-02-22T08:35:01+00:00"

File field:
  file          = <PNG binary>  (field name: "file")

Server must use request.FILES['file'] or equivalent.
Response: HTTP 200/201 with any body.
🎥 Screen Recordings POST multipart/form-data
📋 Demo Format (multipart — NOT JSON) 
Content-Type: multipart/form-data

Form fields:
  pcName          = "DESKTOP-ABC123"
  macAddress      = "00:1B:44:11:3A:B7"
  userName        = "jdoe"
  timestamp       = "2026-02-22T08:00:00+00:00"
  durationSeconds = 300
  syncTime        = "2026-02-22T08:35:01+00:00"

File field:
  file            = <MP4 binary>  (field name: "file")

Note: MP4 chunks are ~5 minutes / ~50-150 MB each.
      Set server upload limit accordingly (e.g. client_max_body_size 200m in nginx).
Response: HTTP 200/201 with any body.
